# 🎮 Connect 4 Game

This project is a browser-based **Connect 4 Game** developed using **HTML, CSS, and JavaScript**. It allows two players to play the classic Connect Four game with real-time turn switching and automatic win detection.

---

## 🚀 Features

* 🔴🟡 Two-player gameplay
* 🎯 Automatic win detection
* 🔄 Restart game functionality
* 🎨 Interactive game board UI
* ⚡ Real-time turn updates
* 📱 Responsive design support

---

## 🛠️ Technologies Used

* JavaScript

## ⚙️ How the Game Works

* The game board contains **6 rows × 7 columns**
* Players take turns dropping discs:

  * 🔴 Red Player
  * 🟡 Yellow Player
* The first player to connect:

  * 4 horizontally
  * 4 vertically
  * 4 diagonally
    wins the game 🎉

## ▶️ How to Run

1. Download or clone the repository
2. Make sure all files are in the same folder
3. Open `index.html` in your browser
4. Start playing 🎮
